## 1. 产品概述

共享文件夹权限审计看板是一款面向中大型企业信息安全团队和部门数据管理员的数据驱动型云盘权限治理工具。它将云盘共享权限转化为可追踪、可量化的安全指标，帮助安全团队实现从被动响应到主动治理的转变。

- 核心价值：将模糊的"共享权限管理"问题转化为可度量、可追踪、可闭环的安全治理流程
- 目标用户：企业信息安全团队、部门数据管理员、IT 运维团队
- 应用场景：日常权限巡检、安全例会汇报、风险整改追踪、合规审计支撑

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 安全管理员 | 信息安全团队成员 | 查看全量数据、分配整改任务、导出报表、配置规则 |
| 部门负责人 | 各业务部门数据管理员 | 查看本部门数据、处理整改任务、标记复核完成 |
| 高管视图 | 管理层 | 查看汇总指标、趋势概览、治理成效 |

### 2.2 功能模块

1. **总览仪表盘**：核心指标卡片、风险趋势图、部门排行、待办事项
2. **风险钻取**：部门/项目/负责人维度下钻、文件夹详情、风险原因分析、历史记录
3. **整改闭环**：任务分配、截止日期设置、整改状态追踪、超期提醒
4. **月度报告**：新增风险、关闭风险、反复问题、治理成效统计
5. **配置管理**：风险规则配置、复核周期设置、通知模板管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 总览仪表盘 | 顶部指标卡 | 外链数量、外部账号数量、高危可编辑目录数量、超期未复核文件夹数量 |
| 总览仪表盘 | 趋势分析图 | 近30天风险数量变化趋势、新增/关闭/存量趋势对比 |
| 总览仪表盘 | 风险排行 | 按部门、项目、负责人三个维度的风险数量排行 |
| 总览仪表盘 | 待办事项 | 待处理整改任务、即将超期任务、今日新增风险 |
| 风险详情页 | 文件夹列表 | 部门下所有共享文件夹列表，支持风险等级筛选 |
| 风险详情页 | 风险详情抽屉 | 风险原因、首次发现时间、历史整改记录、相关人员 |
| 风险详情页 | 整改操作 | 分配负责人、设置截止日期、添加备注、标记完成 |
| 月度报告页 | 数据汇总 | 本月新增/关闭/存量风险、整改率、平均处理时长 |
| 月度报告页 | 反复问题 | 多次出现的风险项、复发率统计 |
| 月度报告页 | 部门对比 | 各部门治理成效对比排名 |

## 3. 核心流程

```mermaid
flowchart LR
    A["安全管理员登录看板"] --> B["查看总览指标与趋势"]
    B --> C["发现高风险部门"]
    C --> D["点击部门下钻查看"]
    D --> E["查看具体文件夹风险详情"]
    E --> F["分配整改任务给负责人"]
    F --> G["设置复核截止日期"]
    G --> H["负责人接收并处理"]
    H --> I["标记整改完成"]
    I --> J["安全管理员复核确认"]
    J --> K["月底自动生成治理报告"]
```

用户日常使用流程：安全管理员登录后首先查看总览仪表盘的核心指标和趋势图，发现风险较高的部门后点击下钻，查看具体共享文件夹的风险详情和历史整改记录，然后将整改任务分配给对应负责人并设置截止日期。负责人处理完成后标记完成，安全管理员进行复核确认。月底系统自动汇总生成治理报告，用于安全例会汇报。

## 4. 用户界面设计

### 4.1 设计风格

- **设计方向**：企业级安全仪表盘风格，专业、严谨、数据驱动
- **主色调**：深海蓝 (#0F172A) 作为主背景，营造专业安全氛围
- **强调色**：
  - 危险红 (#EF4444)：高危风险、超期未处理
  - 警告橙 (#F59E0B)：中风险、即将超期
  - 成功绿 (#10B981)：低风险、已完成
  - 信息蓝 (#3B82F6)：普通信息、外链
- **字体**：
  - 标题：Geist Sans，现代无衬线字体，清晰专业
  - 数据数字：等宽字体，确保数字对齐易读
- **布局风格**：卡片式布局，网格化排列，信息层级分明
- **动效**：数据加载时的数字滚动动画、卡片悬停微交互、下钻时的平滑过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 总览仪表盘 | 顶部指标卡 | 大数字展示、趋势箭头、图标、环比变化百分比 |
| 总览仪表盘 | 趋势分析图 | 面积图、多数据线、时间筛选器、图例 |
| 总览仪表盘 | 风险排行 | 横向柱状图、排行榜样式、可切换维度 |
| 总览仪表盘 | 待办列表 | 列表项、风险标签、截止日期、操作按钮 |
| 风险详情页 | 文件夹列表 | 表格视图、风险等级色标、排序筛选 |
| 风险详情页 | 详情抽屉 | 侧边滑入、时间线、状态标签 |
| 月度报告页 | 数据卡片 | 指标对比、环比箭头、进度环 |

### 4.3 响应式

- 桌面端优先设计，针对 1440px 及以上宽度优化
- 平板端适配：两列布局转单列，图表自适应缩放
- 移动端：纵向滚动布局，核心指标优先展示，表格转卡片视图
- 触摸优化：增大点击区域，支持滑动手势切换标签

### 4.4 数据可视化

- 图表库：使用 ECharts 或 Recharts 实现专业数据可视化
- 图表类型：面积图（趋势）、横向柱状图（排行）、环形图（占比）、进度环（完成率）
- 交互动效：数据加载动画、悬停高亮、点击下钻过渡
- 无障碍：支持键盘导航、图表数据文本化、颜色对比度符合 WCAG AA 标准
